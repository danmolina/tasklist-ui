# Getting Started

#### Clone the repo
```sh
git clone https://github.com/danmolina/tasklist-ui.git
cd tasklist-ui
```

#### Install dependencies:
```sh
npm install
```

#### Run
```sh
npm start
```


# Using Docker
#### Pull Repository
```sh
docker pull danmichaelmolina/tasklist-ui:1.0.0
```
#### Run
```sh
docker run -p 3000:3000 danmichaelmolina/tasklist-ui:1.0.0
```
